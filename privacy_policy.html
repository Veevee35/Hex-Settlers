Privacy Policy

Effective Date: July 14, 2026
Last Updated: July 14, 2026

This policy describes the information processed by Hex Settlers, an online multiplayer browser game. It is intended to explain the application's current behavior; hosting operators should adapt it to their deployment and obtain legal review where required.

1. Information Processed

  • Account information: username, display name, a salted password hash, session-token hashes, account timestamps, and aggregate game statistics.
  • Game information: lobby membership, player colors, chat messages, game state, completed-game history, and postgame statistics.
  • Technical information: ordinary HTTP/WebSocket request data, such as IP address and user agent, may be processed by the server or hosting provider. Client addresses are used temporarily for authentication throttling and are not written to the application's JSON databases.
  • Browser data: interface preferences, the last room code, audio/accessibility settings, and custom texture packs may be stored locally in browser storage or caches.

2. How Information Is Used

Information is used to authenticate players, operate and reconnect multiplayer games, enforce game rules, display chat/history/leaderboards, save preferences, prevent authentication abuse, diagnose failures, and maintain the service. The application contains no advertising or third-party analytics integration.

3. Sessions and Cookies

The browser uses a same-site, HTTP-only session cookie so JavaScript cannot read the persistent bearer credential. The cookie normally lasts up to 30 days. Older browser versions may have stored a session token in local storage; the current application exchanges that token for the protected cookie and removes the old value.

4. Retention

  • Account records remain until the hosting operator removes them.
  • Session records expire after approximately 30 days and a maximum of eight are retained per account.
  • Completed-game history is limited to the most recent 500 games.
  • Active rooms are removed after eight hours without a connected player.
  • Browser preferences and locally imported texture packs remain until the user clears or deletes them.

5. Sharing and Hosting

The application does not sell account or game information. The server's hosting provider and network infrastructure may process technical data to deliver the service. Players in the same room can see display names, public game state, and chat messages. Completed-game information may appear in game history and leaderboards.

6. Security

Passwords are salted and hashed. Server-side session records are hashed, persistent files are written atomically, browser sessions use protected cookies, and game actions are validated by the server. No system can guarantee absolute security; use a unique password and do not place sensitive information in display names or chat.

7. Children

The service is not directed to children under 13. A hosting operator who learns that an underage user supplied personal information should remove the associated account and records as required by applicable law.

8. Changes and Contact

This policy may change as the game evolves. Material changes should be reflected by updating the date above. Questions or deletion requests should be directed to the operator of the Hex Settlers deployment. For the reference project, use the project issue tracker.
